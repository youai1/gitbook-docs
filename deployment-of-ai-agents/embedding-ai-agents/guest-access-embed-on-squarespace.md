# Guest Access Embed on Squarespace

## Overview

Allow anyone to access your AI agent by enabling guest access. This is an easy method to embed  your AI agent.

## Enable Guest Access

Enable guest access to your AI agent.

1. Select the name of your AI agent.
2. Select **Embedding** under **Sharing**.
3. Select **Enable Guest Access**.
4. Copy the code to embed into your website.

## Add to Squarespace

Add the embed code to your Squarespace website.

1. Open your **Squarespace** website.
2. Navigate to **Website** from the left-side navigation, then select **Pages**.
3. Click the **+** button beside **Main Navigation** to add a new blank page.
4. Click the **Edit** button.
5. Click the **Add Section** button.
6. Click **Add a blank section**.
7. Click **+ Add Block**.
8. Select **Code**.
9. Replace the **Hello World!** placeholder with the code you copied from MindStudio into the Code block.
10. Resize the MindStudio embed code to ensure your AI agent fits comfortably (e.g., set width to 100% of the container and height to 600px+).
11. Resize the Code block to accommodate your revised size of your AI agent.
12. Click **Save**. Your AI agent is live.
