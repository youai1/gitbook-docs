# Capture Payment Block

The Capture Payment block captures payment that the user authorized from an [Authorize Payment](authorize-payment-block.md) block in your AI agent. Use immediately preceding the End block of a workflow.

See the [Earnings](https://app.mindstudio.ai/workspace/earnings) page to view pending transactions. Collect payouts using Stripe Connect.

There is no configuration.
